Sobre o Projeto

ViSound: Inclusão Digital através da Acessibilidade

O ViSound é um projeto de Trabalho de conclusão de Curso (TCC) que busca promover a acessibilidade digital para as pessoas com dificiências auditivas e visuais. Através de quatro extensões do navegador, o projeto oferece soluções práticas e eficientes para melhorar a experiência da navegação web.

Cada extensão foi desenvolvida com foco em usabilidadde, performance e compatibilidade, garantindo que pessoas com diferentes tipos de deficiências possam acessar conteúdo digital de forma inclusiva e autônoma.

Objetivos do Projeto

✓ Facilitar o acesso a informação para deficientes auditivos através de tradução para libras;

✓ Disponibilizar leitura de tela para deficientes visuais;

✓ Oferecer ajustes visuais para pessoas com Daltonismo;

✓ Permitir personalização de fontes para pessoas com baixa visão ou dislexia.
