ViSound

O que é o ViSound?

É uma extensão do Google Chrome que tem a finalidade de disponibilizar recursos e ajuda para uma navegação na internet mais acessível e fácil para aqueles que têm dificuldade ou deficiência visual e/ou auditiva. Ela pode mexer no alto contraste e diminuir/aumentar a fonte da letra e mudar a paleta de cores de uma aba para os diferentes tipos de daltonismo — como protanopia, deuteranopia, tritanopia — para ajudar pessoas que sofrem de deficiência visual. Também adiciona uma função de traduzir trechos marcados pelo usuário para Libras por meio de uma janelinha com um avatar que fala em linguagem de sinais, para auxiliar os que possuem deficiência auditiva.
